## 变量和运算符

---

### 1、变量

变量是程序的基本组成单位，当程序/代码执行后，变量的值是存在计算机内存的。

`Memory`（内存）是计算机的重要部件，它用于暂时存放 CPU 中的运算数据，以及与硬盘等外部存储交换的数据。它是外存于 CPU 进行沟通的桥梁，计算机中所有程序的运行都在内存中进行。

变量相当于内存中一个数据存储空间的表示，可以把变量看做是一个房间的门牌号，通过门牌号我看可以找到房间，而通过变量名可以访问到变量（值）。
